# Aplicacion de Mensajes Personalizados

Una aplicacion simple que combina frontend y backend para mostrar un mensaje personalizado cuando el usuario ingresa su nombre

## Requisitos

- Node.js

## Instalacion

1. Clonar el repositorio:
```
vas a la carpeta o lugar donde quieras clonar el repositorio
git  https://github.com/luciannoFr/tlp-III-tp-1.git
cd tlp-III-tp-1
```

2. Instalamos las dependencias:
```
npm i
```

## Ejecucion

1. Iniciar el servidor:
```
node server.js
```

3. Abrimos el navegador y vamos a la url:
```
http://localhost:3000
```
